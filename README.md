## 命名規則

* クラス名  
大文字のやつ  
ex) TestClass  

* メソッド名  
小文字アンダーバーのやつ  
ex) test_method  

* 変数  
小文字アンダーバーのやつ  
ex) test_variable  

* 定数  
全部大文字アンダーバー  
ex) FINAL_STATIC
